# SkinSense UI 設計構思

---

## 1. 設計理念

### 核心理念
**「專業但溫暖，像一位懂皮膚的朋友」**

- 不要過於醫學冷冰
- 不要過於化妝品廣告感
- 要讓用戶感到被理解、被幫助

### 設計關鍵詞
| 關鍵詞 | 說明 |
|--------|------|
| **清新** | 輕盈的感覺，皮膚需要呼吸 |
| **專業** | 讓用戶信任 AI 的建議 |
| **溫暖** | 不冷漠，像朋友在關心你的皮膚 |
| **簡單** | 不復雜，一步一步引導 |

## 2. 色彩系統

### 當前色彩
```css
--skin-50: #FFF5F5     /* 背景 */
--skin-200: #FED7D7    /* 邊框 */
--skin-400: #F6A5A5    /* 次要元素 */
--skin-500: #ED6363    /* 主色 */
--skin-600: #E53E3E    /* 主色深 */
--skin-700: #C53030    /* 主色更深 */
```

### 建議優化

#### 方案 A：「清新花瓣」
```
Primary:    #E8A4B8  (玫瑰粉)
Secondary:  #B8E0D2  (薄荷綠)
Accent:     #F6D87C  (陽光黃)
Background: #FDF8F5  (暖白)
Text:       #2D3748  (深灰)
```

#### 方案 B：「專業護膚」（推薦）
```
Primary:    #6366F1  (靛藍紫 - 專業感)
Secondary:  #10B981  (翡翠綠 - 健康)
Accent:     #F59E0B  (琥珀 - 提醒)
Background: #FAFBFC  (冷白)
Surface:    #FFFFFF  (純白)
Text:       #1F2937  (深灰)
```

#### 方案 C：保留現有，優化細節
保留現有紅色系，增加綠色系表達「健康/改善」

```
--healthy: #10B981   /* 改善、成功 */
--warning: #F59E0B   /* 提醒、衝突 */
--info:    #3B82F6   /* 資訊 */
```

## 3. 字體系統

### 當前
```tsx
fontFamily: sans-serif  // 系統預設
```

### 建議（使用 Google Fonts）

```tsx
// 標題字體
font-family: 'Noto Sans TC', sans-serif;
// 英文搭配
font-family: 'Inter', sans-serif;

// 中文正文
font-family: 'Noto Sans TC', -apple-system, BlinkMacSystemFont, sans-serif;
```

**字體大小階梯：**
| 名稱 | 大小 | 用途 |
|------|------|------|
| xs | 12px | 輔助說明 |
| sm | 14px | 次要文字 |
| base | 16px | 正文 |
| lg | 18px | 強調 |
| xl | 20px | 小標題 |
| 2xl | 24px | 區塊標題 |
| 3xl | 30px | 頁面標題 |
| 4xl | 36px | Hero 標題 |

## 7. 動畫規範

### 全域動畫設置 (tailwind.config.js)

```js
module.exports = {
  theme: {
    extend: {
      animation: {
        // 漸入
        'fade-in': 'fadeIn 0.3s ease-out',
        // 上浮
        'slide-up': 'slideUp 0.3s ease-out',
        // 脈衝（溫和）
        'pulse-subtle': 'pulseSubtle 2s infinite',
      },
      keyframes: {
        fadeIn: {
          '0%': { opacity: '0' },
          '100%': { opacity: '1' },
        },
        slideUp: {
          '0%': { opacity: '0', transform: 'translateY(10px)' },
          '100%': { opacity: '1', transform: 'translateY(0)' },
        },
        pulseSubtle: {
          '0%, 100%': { opacity: '1' },
          '50%': { opacity: '0.8' },
        },
      },
    },
  },
}
```

### 使用方式

```tsx
// 漸入動畫
<div className="animate-fade-in">
  內容
</div>

// 卡片 hover 效果（通过 Tailwind 默認的 hover 實現）
<div className="transition-all duration-200 hover:shadow-lg hover:-translate-y-0.5">
  卡片
</div>
```
